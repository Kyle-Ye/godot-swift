><(lhs:rhs:)

Computes the matrix product of an n × 3 matrix, and a 3 × 3 matrix, returning an n × 3 matrix.

Overview

Note: This operation is vectorized in the vertical direction along its left-hand-operand, which means it is most efficient when the lhs matrix has more rows than columns.
