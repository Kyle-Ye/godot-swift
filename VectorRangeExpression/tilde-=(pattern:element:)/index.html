Operator

~=(pattern:element:)

Returns a boolean value indicating whether a value is included in a vector range.

Declaration

static func ~= (pattern:Self, element:Bound) -> Bool

Parameters

pattern

A vector range.

element

A value to match against pattern.

Return value

true if element is contained in the vector range pattern; otherwise, false.
