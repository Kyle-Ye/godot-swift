Generic Operator

><(lhs:diagonal:)

Computes the matrix product of an n × 4 matrix, and a 4 × 4 diagonal matrix, returning an n × 4 matrix.

Declaration

func >< <Column, T>(lhs:Vector<Column, T>.Matrix4, diagonal:Vector4<T>.Diagonal) -> Vector<Column, T>.Matrix4 where Column:SIMD, Column.Scalar == T, T:BinaryFloatingPoint

Parameters

lhs
diagonal

Overview

The result of this operation is the lhs matrix with its columns scaled by the corresponding elements along the diagonal of diagonal.

Note: This operation is much more efficient than performing a full matrix multiplication.
