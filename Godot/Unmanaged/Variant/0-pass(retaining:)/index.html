Generic Static Method

pass(retaining:)

Returns an unmanaged variant value holding the given value, performing an unbalanced retain.

Declaration

static func pass<T>(retaining value:T) -> Self where T:Godot.VariantRepresentable

Parameters

value

Return value

An unmanaged variant value.

Overview

This method calls the passRetained() instance method on T. Prefer calling this method over calling T.passRetained() directly.
