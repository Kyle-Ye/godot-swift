Instance Method

assign(retaining:)

Deinitializes the value stored in this variant value, and replaces it with the given polymorphic value, performing an unbalanced retain on it.

Declaration

mutating func assign(retaining value:Godot.Variant?)

Parameters

value

A variant existential.

Overview

If value is nil, calling this method is equivalent to calling the statically-typed assign(retaining:) method with an empty tuple argument.
