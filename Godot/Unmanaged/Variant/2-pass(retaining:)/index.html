Static Method

pass(retaining:)

Returns an unmanaged variant value holding the given polymorphic value, performing an unbalanced retain.

Declaration

static func pass(retaining value:Godot.Variant?) -> Self

Parameters

value

A variant existential.

Return value

An unmanaged variant value.

Overview

If value is nil, calling this constructor is equivalent to calling the statically-typed pass(retaining:) constructor with an empty tuple argument.
