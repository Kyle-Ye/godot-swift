Structure

Godot.VariantType

A GDScript core type metatype identifier.

Declaration

struct VariantType:Hashable

Overview

This structure models the Godot::Variant::Type enumeration. It is capable of representing variant metatypes that are not known to Godot Swift, which may occur in situations where Swift nativescripts are run by a different version of the Godot engine runtime than they were built for.
