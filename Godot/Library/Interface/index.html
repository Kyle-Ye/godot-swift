Godot.Library.Interface

A list of nativescript types vended by this Swift library.

Declaration

@resultBuilder struct Interface

Overview

This type is its own result builder, which provides a convenient syntax for specifying library interfaces:

@Interface 
static 
var interface:Interface 
{
    MyExportedSwiftClass.self       <- "MyExportedSwiftClass"
    MyOtherExportedSwiftClass.self  <- "MyOtherExportedSwiftClass"
}

It is acceptable for the same Swift type to be registered to more than one GDScript symbol.
