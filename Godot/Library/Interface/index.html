Structure

Godot.Library.Interface

Specifies the nativescript types vended by this Swift library.

Declaration

@resultBuilder struct Interface

Overview

A library interface consists of zero or more type descriptors. Create a type descriptor using the <-(type:symbol:) binding operator:

MyExportedSwiftClass.self <- "MyExportedSwiftClass"

This type is its own result builder, which provides a convenient syntax for defining library interfaces:

@Interface 
static 
var interface:Interface 
{
    MyExportedSwiftClass.self       <- "MyExportedSwiftClass"
    MyOtherExportedSwiftClass.self  <- "MyOtherExportedSwiftClass"
}

It is acceptable for the same nativescript type to be bound to more than one GDScript class name. The game engine will register nativescripts under all of the class names they are bound to.
