Godot.Library

A type specifying the nativescript types vended by your Swift library.

Declaration

struct Library:Godot.NativeLibrary

Overview

For your Swift library to compile, you must extend this type to provide the required library interface.

extension Godot.Library 
{
    @Interface 
    static 
    var interface:Interface 
    {
        MyExportedSwiftClass.self       <- "MyExportedSwiftClass"
        MyOtherExportedSwiftClass.self  <- "MyOtherExportedSwiftClass"
    }
}
