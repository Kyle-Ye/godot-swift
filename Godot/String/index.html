Structure

Godot.String

An opaque string value.

Declaration

struct String:Godot.Variant, StringRepresentable

Overview

This type corresponds to the Godot::String type in GDScript. It is rarely necessary to use Godot.String directly in user code, as Swift’s native String type conforms to VariantRepresentable and StringRepresentable.

This type is an opaque wrapper around a Godot::String object, and it supports no functionality other than converting to and from a native Swift String (or any other StringRepresentable type). The main purpose of this type is to allow strings and variant existentials to be moved without copying the underlying string buffer, in situations where it is not necessary to interact with the actual contents of the string.

Convert between String and Godot.String using the init(_:)’s on each type:

let godot:Godot.String = ...
let swift:Swift.String = .init(godot)
let swift:Swift.String = ...
let godot:Godot.String = .init(swift)

Instances of Godot.String are memory-managed by Swift.
