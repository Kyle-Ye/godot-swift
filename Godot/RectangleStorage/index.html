Protocol

Godot.RectangleStorage

A unification protocol providing support for converting qualifying instances of Vector.Rectangle and Vector.ClosedRectangle to and from their GDScript representations.

Declaration

protocol RectangleStorage:Godot.VectorStorage

Overview

All types conforming to VectorFiniteRangeExpression support the functionality required to conform to Godot.VariantRepresentable, as long as Bound.Storage conforms to this protocol.

Conditional conformances to Godot.VariantRepresentable have already been provided for Vector.Rectangle and Vector.ClosedRectangle. You can add support for additional VectorFiniteRangeExpression types by explicitly declaring the conformance to Godot.VariantRepresentable in user code.

Conforming types

SIMD2

When Scalar conforms to Godot.RectangleElement.

SIMD3

When Scalar conforms to Godot.RectangleElement.
