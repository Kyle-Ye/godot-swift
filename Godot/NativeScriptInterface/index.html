Godot.NativeScriptInterface

Specifies the signals, instance methods, and instance properties vended by T.

Declaration

@resultBuilder struct NativeScriptInterface<T> where T:Godot.NativeScript

Overview

A nativescript interface consists of zero or more signal descriptors, method descriptors, or property descriptors.
