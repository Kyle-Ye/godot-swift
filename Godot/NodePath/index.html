Class

Godot.NodePath

A pre-parsed scene tree path.

Declaration

final class NodePath:Godot.Variant

Overview

This type corresponds to the Godot::NodePath type in GDScript.

Create a NodePath instance from a string using the init(parsing:) initializer.

let string:Swift.String = ...
let path:Godot.NodePath = .init(parsing: string)

Instances of NodePath are memory-managed by Swift.
