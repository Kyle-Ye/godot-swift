Initializer

init(retained:)

Unsafely creates an instance of this class from a raw delegate pointer.

Declaration

required init(retained core:UnsafeMutableRawPointer)

Parameters

core

Overview

The core pointer is assumed to have been properly type-checked. If Self is Godot.AnyObject, or one of its subclasses, it is also assumed to have already been retained, and will be released when self is deinitialized.

Do not use this initializer unless you really know what you are doing.
