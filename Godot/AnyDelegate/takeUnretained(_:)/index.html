Static Method

takeUnretained(_:)

Attempts to load an instance of Self from a variant value.

Implements requirement in Godot.Variant.

Declaration

static func takeUnretained(_ value:Unmanaged.Variant) -> Self?

Parameters

value

Return value

An instance of Self, if value held a delegate whose dynamic type was a subclass of Self; nil otherwise.

Warning

The static type of the return value is Self, but its dynamic type may actually be one of its subclasses. You can query its dynamic type with the is and as operators, just like any other Swift class.
