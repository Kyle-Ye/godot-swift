Static Method

register(as:with:)

Registers this nativescript type with the Godot engine under the given GDScript class names.

Required. Synthesized.

Declaration

static func register(as symbols:[String], with library:Godot.Library)

Parameters

symbols

The GDScript class names to register this nativescript as.

library

Overview

Do not manually implement this method for types declared in the library Interface. However, it may be useful to define an implementation stub for this method while prototyping.
