Protocol

Godot.NativeLibrary

Declares the requirements that Godot.Library must satisfy.

Declaration

protocol NativeLibrary

Overview

For your Swift library to compile, you must extend the provided Godot.Library type to provide the required interface.

Conforming additional types to this protocol has no effect.

Conforming types

Godot.Library
