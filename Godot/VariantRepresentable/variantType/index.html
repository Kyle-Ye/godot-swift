Static Property

variantType

A type hint used by GDScript.

Required.

Default implementation provided when Self conforms to FixedWidthInteger.

Default implementation provided when Self conforms to BinaryFloatingPoint.

Default implementation provided when Self conforms to VectorFiniteRangeExpression and Self.Storage conforms to Godot.RectangleStorage.

Declaration

static var variantType:Godot.VariantType { get }

Overview

If a conforming type can be represented by more than one GDScript type, this property should return VariantType.void.
