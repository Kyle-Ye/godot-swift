Generic Class

Godot.Array

One of the Godot pooled array types.

Declaration

final class Array<Element>:Godot.Variant where Element:Godot.ArrayElement

Overview

If Element is String, this type corresponds to the Godot::PoolStringArray type.

If Element is UInt8, this type corresponds to the Godot::PoolByteArray type.

If Element is Int32, this type corresponds to the Godot::PoolIntArray type.

If Element is Float32, this type corresponds to the Godot::PoolRealArray type.

If Element is Vector2<Float32>, this type corresponds to the Godot::PoolVector2Array type.

If Element is Vector3<Float32>, this type corresponds to the Godot::PoolVector3Array type.

If Element is Vector4<Float32>, this type corresponds to the Godot::PoolColorArray type.
