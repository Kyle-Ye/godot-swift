Class

Godot.ResourceIdentifier

A unique identifier for a Resource.

Declaration

final class ResourceIdentifier:Hashable, Godot.Variant

Overview

This type corresponds to the Godot::RID type in GDScript.

Create a ResourceIdentifier from a raw Int bit-pattern using the memberwise init(bitPattern:) initializer. Only use this if you really know what you are doing.

let bits:Int = ...
let rid:Godot.ResourceIdentifier = .init(bitPattern: bits)

Warning

Godot resource identifiers are semantically similar to opaque pointers, and resource identifiers with invalid bit-patterns may cause runtime crashes.

The ResourceIdentifier type is a trivial type, and therefore does not require memory management.
