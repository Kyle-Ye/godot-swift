Generic Instance Method

take(unretained:at:)

Attempts to load the element at the given index as the specified type.

Declaration

func take<T>(unretained type:T.Type, at index:Int) -> T? where T:VariantRepresentable

Parameters

type
index

The index of the element to access, which must be in the valid index range of this list. Passing an out-of-range index is an error.

Return value

An instance of T, if it was possible to load it from the variant value at index; otherwise nil.
