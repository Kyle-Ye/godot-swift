Generic Initializer

init(_:)

Initializes a list from the given variants.

Declaration

convenience init<C>(_ elements:C) where C:Collection, C.Element == Godot.Variant?

Parameters

elements

A collection containing the elements to form the list from.

Overview

After calling this initializer, each variant element is retained to reflect the fact that this list instance now holds a strong reference to it. This does not affect variant elements holding unmanaged values, such as Unmanaged.Node instances.
