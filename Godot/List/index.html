Class

Godot.List

A heterogenous collection of variant values.

Declaration

final class List:Godot.Variant, RandomAccessCollection, MutableCollection, ExpressibleByArrayLiteral

Overview

This type corresponds to the Godot::Array type in GDScript. It has reference semantics, so modifications to an instance of List will be seen by all values referring to it.

Instances of this type are memory-managed by Swift.
