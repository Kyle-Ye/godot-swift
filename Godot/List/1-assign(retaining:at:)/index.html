Instance Method

assign(retaining:at:)

Assigns a Void value to the list element at the given index.

Declaration

func assign(retaining value:Void, at index:Int)

Parameters

value

An empty tuple.

index

The index of the element to access, which must be in the valid index range of this list. Passing an out-of-range index is an error.

Overview

This method is equivalent to assigning nil through the [_:] subscript, but is slightly more efficient.
