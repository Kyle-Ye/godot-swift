Instance Method

take(unretained:at:)

Attempts to load the element at the given index as an instance of Void.

Declaration

func take(unretained type:Void.Type, at index:Int) -> Void?

Parameters

type
index

The index of the element to access, which must be in the valid index range of this list. Passing an out-of-range index is an error.

Return value

An instance of Void, if the variant value at index was a null variant; otherwise nil.
