Generic Initializer

init(moving...:)

Move-initializes a list from the given unmanaged variants.

Declaration

convenience init<C>(moving elements:Unmanaged.Variant...)

Parameters

elements

Overview

This initializer transfers ownership of the variant elements to the newly-initialized list instance. When the list is deinitialized, all of its elements will be deinitialized as well, which means that the original unmanaged variant values will no longer be valid.
