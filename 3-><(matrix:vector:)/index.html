><(matrix:vector:)

Computes the matrix product of an n × 3 matrix, and an 3-element column vector, returning an n-element column vector.

Declaration

func >< <Column, T>(matrix:Vector<Column, T>.Matrix3, vector:Vector3<T>) -> Vector<Column, T> where Column:SIMD, Column.Scalar == T, T:FixedWidthInteger

Parameters

matrix
vector

Overview

Note: This operation is vectorized in the vertical direction, which means it is most efficient when the matrix type has more rows than columns.
