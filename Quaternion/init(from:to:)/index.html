Initializer

init(from:to:)

Creates a quaternion representing a 3D rotation between two points on the unit sphere.

Declaration

init(from start:Vector3<T>, to end:Vector3<T>)

Parameters

start

A unit vector representing the starting point of a 3D rotation.

end

A unit vector representing the ending point of a 3D rotation.

Overview

Both start and end should be unit vectors (with norm equal to 1.0) for the result to be meaningful.
