Initializer

init(axis:angle:)

Creates a quaternion representing a 3D rotation of the given angle about the given axis.

Declaration

init(axis:Vector3<T>, angle:T)

Parameters

axis

A unit vector specifying a rotation axis.

angle

The rotation angle, in radians.

Overview

The axis vector should be a unit vector (with norm equal to 1.0) for the result to be meaningful.
