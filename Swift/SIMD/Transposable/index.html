Protocol

SIMD.Transposable

An SIMD backing storage type which supports arbitrary elementwise transformations.

Declaration

protocol Transposable:SIMD

Overview

You can conform additional types to this protocol to add linear algebra support for 8-, 16-, etc. dimensional vectors. Only do this if you really know what you are doing.
