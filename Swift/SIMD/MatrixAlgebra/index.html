Protocol

SIMD.MatrixAlgebra

An SIMD backing storage type which supports computing the determinant and inverse of an appropriately-sized matrix type.

Declaration

protocol MatrixAlgebra:SIMD.Transposable

Overview

You can conform additional types to this protocol to add linear algebra support for additional matrix sizes. Only do this if you really know what you are doing.

Conforming types

SIMD2

When Scalar conforms to BinaryFloatingPoint.

SIMD3

When Scalar conforms to BinaryFloatingPoint.
