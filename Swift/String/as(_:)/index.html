as(_:)

Returns this string as a Godot string.

Implements requirement in Godot.StringRepresentable.

Declaration

func as(_ type:Godot.String.Type) -> Godot.String

Parameters

type

Overview

This method copies the storage of this string, so the returned Godot string is completely independent of the original Swift string.

Avoid calling this method directly; using the generic Godot.String.init(_:) initializer is the preferred form.
