Instance Method

as(_:)

Returns a Godot pooled array containing the elements of this Swift array.

Implements requirement in Godot.ArrayRepresentable.

Available when Element conforms to Godot.ArrayElement.

Declaration

func as(_ type:Godot.Array<Element>.Type) -> Godot.Array<Element>

Parameters

type

Overview

This method copies the storage of this array. Because all valid ArrayElement types are trivial value types, the returned pooled array is completely independent of the original Swift array.

Avoid calling this method directly; using the generic Godot.Array.init(_:) initializer is the preferred form.
