Initializer

init(_:)

Creates a native Swift array from a Godot pooled array.

Implements requirement in Godot.ArrayRepresentable.

Available when Element conforms to Godot.ArrayElement.

Declaration

init(_ array:Godot.Array<Element>)

Parameters

array

A Godot pooled array.

Overview

This initializer copies the storage of array. Because all valid ArrayElement types are trivial value types, the newly-initialized Swift array is completely independent of the original pooled array.
