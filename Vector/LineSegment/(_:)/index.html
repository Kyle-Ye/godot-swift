(_:)

Interpolates between the endpoints of this line segment by the given parameter.

Declaration

func callAsFunction(_ t:T) -> Vector<Storage, T>

Parameters

t

The interpolation parameter. It is acceptable to pass values less than 0, or greater than 1.

Return value

A vector obtained by linearly interpolating the endpoints of this line segment by t.

Overview

Calling this method is equivalent to writing the code (self.start * (1 - t)).addingProduct(self.end, t).
