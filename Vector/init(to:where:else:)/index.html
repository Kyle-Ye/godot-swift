Initializer

init(to:where:else:)

Creates a vector instance with one of the two given scalar values repeated in all elements, depending on the given mask.

Available when T conforms to AdditiveArithmetic.

Declaration

init(to value:T, where mask:Mask, else empty:T)

Parameters

value

The scalar value to use where mask is set.

mask

The vector mask used to choose between value and empty.

empty

The scalar value to use where mask is clear.

The default value is AdditiveArithmetic.zero.
