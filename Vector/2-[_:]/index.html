Generic Subscript

[_:]

Loads the specified vector swizzle from this vector.

Available when Storage is SIMD2<T>.

Available when Storage is SIMD3<T>.

Available when Storage is SIMD4<T>.

Declaration

subscript<Index>(swizzle:VectorSwizzle2) -> Vector2<T> where Index:FixedWidthInteger & SIMDScalar { get }

Parameters

swizzle

The vector swizzle to extract.

It is acceptable for the vector swizzle to specify elements that are not present in Self, in which case, this vector instance is interpreted as an infinite-length vector with the missing elements filled in by cycling through the existing elements.

For example, if Self is Vector3<T>, then the swizzle ww refers to the same elements as xx.

Overview

This subscript is shorthand for passing the selector value of swizzle to the [_:] subscript.

Note

This subscript will never trap, even if the specified vector swizzle references elements not present in Self.
