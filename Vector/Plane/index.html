Structure

Vector.Plane

An n-dimensional plane.

Available when Storage conforms to SIMD.Extendable and T conforms to BinaryFloatingPoint.

Conforms to Godot.VariantRepresentable when Storage is SIMD3<T>.

Conforms to Godot.Variant when Storage is SIMD3<Float32>.

Declaration

struct Plane:Hashable

Overview

When Storage is SIMD3<Float32>, this type corresponds to the Godot::Plane type in GDScript.

Create a plane from a normal vector and a perpendicular distance from the origin with the init(normal:bias:) initializer:

let normal:Vector3<T>       = ... ,
    bias:T                  = ... 
let plane:Vector3<T>.Plane  = .init(normal: normal, bias: bias)

Access the normal vector and distance from the origin through the normal and bias properties:

let plane:Vector3<T>.Plane  = ...
let normal:Vector3<T>       = plane.normal, 
    bias:T                  = plane.bias

You can convert between floating point precisions with the init(_:) initializer:

let float64:Vector3<Float64>.Plane = ... 
let float32:Vector3<Float32>.Plane = .init(float64)

Getting vector representations of a plane

var bias:T

The perpendicular distance of this plane from the origin.

var normal:Vector<Storage, T>

The normal vector specifying the direction this plane faces.

let affine:Vector<Storage.Extended, T>

The affine vector representation of this plane. The first n elements are its normal, and the last element is its bias.
