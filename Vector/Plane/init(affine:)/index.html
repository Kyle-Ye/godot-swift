Initializer

init(affine:)

Creates a plane from an affine vector.

Declaration

init(affine:Vector<Storage.Extended, T>)

Parameters

affine

An (n + 1)-dimensional vector whose first n components contain a normal vector, and whose last component specifies a distance from the origin.

For some plane operations to be meaningful, the normal vector to be extracted from this parameter should be of unit length.
