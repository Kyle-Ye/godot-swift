Generic Initializer

init(clamping:)

Converts an integer vector with elements of type U to an integer vector with elements of type T, with each element clamped to the range of values representable by T.

Available when Storage is SIMD4<T> and T conforms to FixedWidthInteger.

Declaration

init<U>(clamping other:Vector4<U>) where U:FixedWidthInteger
