Operator

!~(lhs:radius:)

Returns a boolean value indicating if the given vector is outside of the sphere with the given radius, centered on the origin, or on the sphere boundary.

Available when T conforms to BinaryFloatingPoint.

Declaration

static func !~ (lhs:Self, radius:T) -> Bool

Parameters

lhs

A vector to test for sphere membership.

radius

The radius of the sphere to test for membership in.

Return value

true if lhs is outside the specified sphere, or on its boundary; false otherwise.

Overview

Calling this operator is more efficient than taking the norm of lhs and comparing it with radius.

Note: Due to floating point precision error, this operator may still indicate points on the sphere boundary as being inside the sphere. Consider adding a small epsilon-margin to radius to account for this.
