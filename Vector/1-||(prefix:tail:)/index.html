Operator

||(prefix:tail:)

Appends the given scalar value to the given 3-element vector instance, returning a 4-element vector.

Available when Storage is SIMD3<T>.

Declaration

static func || (prefix:Self, tail:T) -> Vector4<T>

Parameters

prefix
tail

Return value

A 4-element vector with prefix in its first 3 positions, and tail in its last position.
