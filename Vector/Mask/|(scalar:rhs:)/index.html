Operator

|(scalar:rhs:)

Returns the result of a bitwise or operation on the vector mask obtained by broadcasting the given boolean value, and the given vector mask.

Declaration

static func | (scalar:Bool, rhs:Self) -> Self

Parameters

scalar

Return value

A vector mask where each element contains the bitwise result of scalar or the corresponding element of rhs.
