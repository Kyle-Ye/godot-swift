Static Method

>=(lhs:scalar:)

Returns a vector mask with the result of a elementwise greater than or equal to comparison of a vector with a scalar value.

Available when T conforms to Comparable.

Declaration

static func >=(lhs:Self, scalar:T) -> Mask

Parameters

lhs
scalar

Return value

A vector mask where each element is set if the corresponding element of lhs is greater than or equal to scalar.
