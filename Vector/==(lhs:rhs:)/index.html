Operator

==(lhs:rhs:)

Returns a vector mask with the result of a elementwise equal to comparison of two vectors.

Available when T conforms to Equatable.

Declaration

static func == (lhs:Self, rhs:Self) -> Mask

Parameters

lhs
rhs

Return value

A vector mask where each element is set if the corresponding element of lhs is equal to the corresponding element of rhs.
