Operator

||(prefix:tail:)

Appends the given scalar value to the given n-element vector instance, returning an (n + 1)-element vector.

Available when Storage conforms to SIMD.Extendable.

Declaration

static func || (prefix:Self, tail:T) -> Vector<Storage.Extended, T>

Parameters

prefix
tail

Return value

An (n + 1)-element vector with prefix in its first n positions, and tail in its last position.
