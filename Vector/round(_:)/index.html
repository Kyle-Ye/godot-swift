Instance Method

round(_:)

Rounds each element of this vector according to the given rounding rule.

Available when T conforms to BinaryFloatingPoint.

Declaration

mutating func round(_ rule:FloatingPointRoundingRule)

Parameters

rule

The rounding rule to use. The default value is FloatingPointRoundingRule.toNearestOrAwayFromZero.
