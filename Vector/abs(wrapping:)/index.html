abs(wrapping:)

Performs an elementwise absolute value operation on the given vector, with twoâ€™s-complement wraparound if the resulting elements are not representable by T.

Available when T conforms to SignedInteger and FixedWidthInteger.

Parameters

value

A vector.

Return value

A vector where each element contains the absolute value of the corresponding element of value, or T.min if the original element was T.min.

Overview

Note: Elements with the value T.min will remain T.min, as the value -T.min is equivalent to T.max when truncated to the bit width of T. All other values are transformed according to the mathematical definition of absolute value.

Note: To obtain the scalar magnitude of an integer vector, convert it to floating point vector, and use the norm instance property.
