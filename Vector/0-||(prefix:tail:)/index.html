Operator

||(prefix:tail:)

Appends the given scalar value to the given 2-element vector instance, returning a 3-element vector.

Available when Storage is SIMD2<T>.

Declaration

static func || (prefix:Self, tail:T) -> Vector3<T>

Parameters

prefix
tail

Return value

A 3-element vector with prefix in its first 2 positions, and tail in its last position.
