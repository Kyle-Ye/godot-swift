Operator

~<(lhs:radius:)

Returns a boolean value indicating if the given vector is contained by the sphere with the given radius, centered on the origin, not including the sphere boundary.

Available when T conforms to BinaryFloatingPoint.

Declaration

static func ~< (lhs:Self, radius:T) -> Bool

Parameters

lhs

A vector to test for sphere membership.

radius

The radius of the sphere to test for membership in.

Return value

true if lhs is strictly inside the specified sphere; false otherwise.

Overview

Calling this operator is more efficient than taking the norm of lhs and comparing it with radius.
