Static Method

sin(_:)

Returns the elementwise sin of the given vector.

Available when Storage conforms to SIMD.Transposable and T conforms to Numerics.Real.

Declaration

Parameters

vector

Return value

Overview

Note: This function is not SIMD-vectorized; it is implemented through scalar operations.
