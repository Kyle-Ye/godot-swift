Static Method

abs(_:)

Performs an elementwise absolute value operation on the given vector.

Available when T conforms to BinaryFloatingPoint.

Declaration

static func abs(_ value:Self) -> Self

Parameters

value

A vector.

Return value

A vector where each element contains the absolute value of the corresponding element of value, or T.nan if the original element was T.nan.

Overview

Note

To obtain the scalar magnitude of a vector, use the norm instance property.
