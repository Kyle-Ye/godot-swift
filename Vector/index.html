Generic Structure

Vector

An SIMD-backed vector.

Conforms to Godot.ArrayElement when Storage conforms to Godot.ArrayElementStorage.

Conforms to Godot.VariantRepresentable when Storage conforms to Godot.VectorStorage.

Conforms to Godot.Variant when Storage conforms to Godot.VectorStorage and T is Float32.

Declaration

struct Vector<Storage, T>:Hashable, CustomStringConvertible where Storage:SIMD, T:SIMDScalar, T == Storage.Scalar

Overview

Create a vector from a tuple literal using the postfix *(row:) operator:

let x:Vector4<Int> = (1, 2, 3, 4)*

Matrices are represented by tuples of vectors, where each vector forms a column of the matrix:

let a:Vector3<Double>           = (1.0, 2.0, 3.0)*,
    b:Vector3<Double>           = (3.0, 2.0, 1.5)*,
    c:Vector3<Double>           = (2.0, 0.5, 2.0)*,
    d:Vector3<Double>           = (0.0, 1.0, 2.0)*

let A:Vector3<Double>.Matrix4   = (a, b, c, d)

You can extend vectors using the ||(prefix:tail:) operator, and perform matrix multiplications using one of the >< operators.

let w:Vector3<Double>   = (0.5, 0.75, 1.0)* 
let y:Vector3<Double>   =  A >< (w || 1.0)
