Generic Operator

<-(method:symbol:)

Binds a method to a GDScript method name. This operator is a variadic-generic template. Godot Swift will synthesize overloads for arbitrary numbers and patterns of generic parameters as needed.

Declaration

func <- <T, __, _>(method:@escaping (T) -> (T.Delegate, __) -> _, symbol:String) -> Godot.NativeScriptInterface<T>.Method where T:Godot.NativeScript, __:Godot.VariantRepresentable, _:Godot.VariantRepresentable

Parameters

method

A curried function, which can generate an instance method given an instance of T.

symbol

The name to expose the method as in GDScript.
