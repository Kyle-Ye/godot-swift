init(_:rounding:)

Converts a finite floating point vector range with bounds with elements of type U to a finite integer vector range with bounds with elements of type T, with each element of each bound rounded according to the given floating point rounding rule.

Available when Storage is SIMD3<T> and T conforms to FixedWidthInteger.
